Client Updater : C++

Client Updater is a library built to create a modular updating system, Client Updater goes through and collects each file in a directory, once it has collected every file Client Updater bundles it up into a file full of hash values, this file is then sent to a server where the same thing has happened, Client Updater then compares the two and sends the client the files it is missing and a message telling the client which files to delete

Easy Net : Unity3D : Work In Progress

Easy Net is a unity editor plugin, that is designed to make Unity networking easier, the way it works if by hiding all of the unity network code in the background while only exposing the user to functions that they create, the background network code then calls the functions that the user makes
